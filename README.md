## Project Objective
An interactive Desktop system for connecting the admin with volunteers, organizing and encouraging them to perform voluntary services.

## Project Highlights
- Admin dashboard for managing volunteers and voluntary services.
- Volunteer dashboard for browsing, joining, and completing services.
- Volunteer service history of their completed voluntary services.
- Real-time notification system for latest updates from the admin.
- Earn XP to level-up volunteer profile to unlock in-app rewards.

## Tech Stack
- **Front-end:** JavaFX, FXML, CSS
- **Back-end:** Java (JDK 11+)
- **Database:** Microsoft Access
- **Architecture:** MVC Pattern

## Project Specifications
- **Data Management:** Real-time CRUD operations for admin to manage volunteers, voluntary services and application requests.
- **Relational Integrity:** This system actively uses database schema, keeping consistency across different tables.
- **Secure Authentication:** We designed a dedicated log-in portal to verify user credentials after having signed-in.
- **Dynamic Search Engine:** The admin can look up for volunteers, using their name or their current voluntary service(s).

## Technical Decisions
- Our team integrated this project using MS Access to demonstrate our ability in handling relational databases efficiently.
- This project follows a modular structure, separating between front-end and back-end for maintainability and user testing.
